# 7-Day Micro-Protocols Library

This library translates behavior science into small, realistic 7-day protocols for busy adults.

Design principles used throughout:
- Small enough to do on stressed days.
- Clear cues (`if X, then Y`) and self-monitoring.
- Non-shaming recovery after misses.
- Progress = consistency and recovery speed, not perfection.

## Evidence Snapshot (Why These Work)
- **Implementation intentions** (`if-then` plans) reliably improve follow-through versus intentions alone (Gollwitzer & Sheeran, 2006).
- **Tiny repeated actions** can become more automatic over time; habit strength grows with repetition and does not require perfect streaks (Lally et al., 2010).
- **Self-monitoring** is a strong active ingredient across health behavior interventions, especially when paired with other self-regulation steps (Michie et al., 2009; Harkin et al., 2016).
- **Relapse prevention** research supports treating lapses as data and using immediate recovery actions to reduce escalation (Marlatt & Gordon model; Witkiewitz & Marlatt review).
- **Digital micro-interventions/JITAIs** support delivering the smallest useful prompt at the right time and context (Nahum-Shani et al., 2018).

## Pillar 1: Do Hard Things

### 1) Two-Minute Start Rep
- **When to recommend:** Procrastination, overwhelm, fear of starting.
- **Minimum:** Start one avoided task for 2 minutes daily.
- **Standard:** 2-minute start + continue to 8 minutes if momentum appears.
- **Advanced:** One morning and one afternoon start rep.
- **Failure recovery:** If missed, do a 60-second start before bed (no catch-up).
- **Daily check-in:** "Did I start one thing before I felt ready?"
- **7-day progress:** Lower start resistance; more tasks moved from "stuck" to "in motion."

### 2) One Hard Thing Before Noon
- **When to recommend:** Reactive mornings, hard tasks postponed all day.
- **Minimum:** Define one "hard thing" and touch it for 3 minutes before noon.
- **Standard:** 15-minute focused block before noon.
- **Advanced:** 25-minute block plus one sentence of next-step planning.
- **Failure recovery:** If noon passes, do a 5-minute block at first next break.
- **Daily check-in:** "Did I pay the hard cost early today?"
- **7-day progress:** Less dread by evening; improved control of day trajectory.

### 3) Discomfort Timer
- **When to recommend:** Avoidance loops and relief scrolling.
- **Minimum:** 5 minutes on the uncomfortable task with timer.
- **Standard:** 10 minutes + 2-minute pause + optional second 10.
- **Advanced:** 15 minutes, no app-switching.
- **Failure recovery:** Restart with 3-minute timer immediately after noticing escape.
- **Daily check-in:** "Did I stay with discomfort for one timer?"
- **7-day progress:** Better distress tolerance; fewer instant escapes.

### 4) First Step Card
- **When to recommend:** "I don't know where to start" paralysis.
- **Minimum:** Write the first physical action for tomorrow's hard task.
- **Standard:** Write cue + first action + fallback action.
- **Advanced:** Add time/place implementation intention.
- **Failure recovery:** Rewrite a smaller first step (under 60 seconds).
- **Daily check-in:** "Did I define the first step before I needed motivation?"
- **7-day progress:** Faster starts; less cognitive friction at go-time.

### 5) Approach Before Escape
- **When to recommend:** Phone-check reflex during stress.
- **Minimum:** One rule: before opening distracting app, do 1 useful minute.
- **Standard:** 3 useful minutes before distraction.
- **Advanced:** 5 useful minutes + log trigger.
- **Failure recovery:** If distracted first, do one useful minute right after.
- **Daily check-in:** "Did I approach once before I escaped?"
- **7-day progress:** Noticeable break in autopilot avoidance.

### 6) Friction Flip
- **When to recommend:** Willpower dependence.
- **Minimum:** Make hard task easier by one environment tweak.
- **Standard:** Add one friction to distraction + one friction removed from task.
- **Advanced:** Rebuild desk/app layout nightly for next day.
- **Failure recovery:** If setup failed, do a 2-minute reset now.
- **Daily check-in:** "Did I make the right action easier than the wrong one?"
- **7-day progress:** More follow-through with less effortful self-control.

### 7) Brave Email/Message Rep
- **When to recommend:** Avoided communication, delayed decisions.
- **Minimum:** Draft and send one avoided message in 5 lines.
- **Standard:** Send one high-friction message by 3 PM.
- **Advanced:** Send + propose next step and timeline.
- **Failure recovery:** If unsent, send a shorter "holding" version now.
- **Daily check-in:** "Did I send one message I was avoiding?"
- **7-day progress:** Reduced background anxiety from open loops.

### 8) Tiny Exposure Ladder
- **When to recommend:** Repeated fear of one specific task.
- **Minimum:** Do level-1 exposure for 2 minutes daily.
- **Standard:** Level-1 for 3 days, level-2 for next 4.
- **Advanced:** Add level-3 once this week.
- **Failure recovery:** Drop one ladder level and complete that today.
- **Daily check-in:** "Did I do today's smallest brave step?"
- **7-day progress:** Fear still present but less controlling.

### 9) Hard-Thing Pairing
- **When to recommend:** Low initiation energy.
- **Minimum:** Pair hard task with neutral cue (same mug, same song, same seat).
- **Standard:** Same cue + fixed start time window.
- **Advanced:** Cue + start script + timer.
- **Failure recovery:** If cue missed, trigger with backup cue (phone alarm label).
- **Daily check-in:** "Did I run my hard-thing cue today?"
- **7-day progress:** More automatic starts at cue time.

### 10) Daily Repair Rep
- **When to recommend:** All-or-nothing collapse after one miss.
- **Minimum:** After any miss, do a 2-minute repair action same day.
- **Standard:** Repair within 30 minutes of noticing miss.
- **Advanced:** Repair + write one prevention tweak.
- **Failure recovery:** If day is over, do repair at next day's first opportunity.
- **Daily check-in:** "Did I repair quickly after a miss?"
- **7-day progress:** Shorter lapse duration; stronger self-trust.

## Pillar 2: The Night Before

### 1) 20-Minute Landing Window
- **When to recommend:** Bedtime drift and mental overactivation.
- **Minimum:** 20 minutes of lower-light, lower-stimulation wind-down.
- **Standard:** Same start time window (+/- 30 min) nightly.
- **Advanced:** Add calming sequence (wash, stretch, read, lights low).
- **Failure recovery:** If late, do a 5-minute mini-landing instead of skipping.
- **Daily check-in:** "Did I give my brain a landing window?"
- **7-day progress:** Easier sleep onset and calmer transition to bed.

### 2) Tomorrow Top-3 Offload
- **When to recommend:** Rumination about unfinished work.
- **Minimum:** Write top 3 for tomorrow on paper.
- **Standard:** Top 3 + first step for #1.
- **Advanced:** Add estimated start time for #1.
- **Failure recovery:** If forgotten, do a 60-second list in bed notebook.
- **Daily check-in:** "Did I offload tomorrow before trying to sleep?"
- **7-day progress:** Fewer looping thoughts at lights-out.

### 3) Phone Parking Rule
- **When to recommend:** Doomscrolling in bed.
- **Minimum:** Charge phone outside arm's reach.
- **Standard:** Charge outside bedroom most nights.
- **Advanced:** Enable app limits + grayscale after parking time.
- **Failure recovery:** If phone re-enters bed, park it again immediately.
- **Daily check-in:** "Did my phone sleep away from me?"
- **7-day progress:** Less bedtime screen creep; faster sleep attempts.

### 4) Last Caffeine Fence
- **When to recommend:** Long sleep latency or light sleep.
- **Minimum:** Set caffeine cutoff 8 hours before target bedtime.
- **Standard:** Track last caffeine time daily.
- **Advanced:** Move cutoff earlier on poor-sleep days.
- **Failure recovery:** If cutoff broken, switch to decaf and protect wind-down.
- **Daily check-in:** "Did I respect today's caffeine fence?"
- **7-day progress:** Improved evening sleepiness signal.

### 5) Last Meal Timing Buffer
- **When to recommend:** Heavy late eating and restless nights.
- **Minimum:** Finish last full meal 2.5 hours before bed.
- **Standard:** 3-hour buffer most nights.
- **Advanced:** Lighter late snack protocol if dinner is delayed.
- **Failure recovery:** If eating late, reduce portion and take gentle 10-minute walk.
- **Daily check-in:** "Did I leave enough buffer after my last meal?"
- **7-day progress:** Less nighttime discomfort; smoother sleep onset.

### 6) Evening Light Dimmer
- **When to recommend:** Alertness spikes late evening.
- **Minimum:** Dim room lights 60 minutes pre-bed.
- **Standard:** Dim lights + warm screen mode.
- **Advanced:** Keep bright overhead lights off after dimmer time.
- **Failure recovery:** If bright exposure occurs, switch back to low light immediately.
- **Daily check-in:** "Did my evening light match nighttime goals?"
- **7-day progress:** Better circadian signaling and bedtime readiness.

### 7) Clothes + Breakfast Prep
- **When to recommend:** Stressful mornings affecting nighttime anxiety.
- **Minimum:** Lay out one morning item (clothes or breakfast tool).
- **Standard:** Prepare clothes + simple breakfast plan.
- **Advanced:** Full 5-minute morning setup station.
- **Failure recovery:** If skipped, prepare one item before sleep.
- **Daily check-in:** "Did I make tomorrow morning easier tonight?"
- **7-day progress:** Lower morning chaos, improved wake consistency.

### 8) Worry Container
- **When to recommend:** Night worry loops.
- **Minimum:** 3-minute "worry write" then close notebook.
- **Standard:** Worry list + one next action per item.
- **Advanced:** Add "not tonight" statement and revisit time tomorrow.
- **Failure recovery:** If rumination returns, write one line and pause.
- **Daily check-in:** "Did I place worries in a container, not my pillow?"
- **7-day progress:** Reduced cognitive spillover into bedtime.

### 9) Bedtime Guardrail
- **When to recommend:** Variable bedtime by 2+ hours.
- **Minimum:** Keep bedtime within a 60-minute window.
- **Standard:** Within 45 minutes.
- **Advanced:** Within 30 minutes on 5+ nights.
- **Failure recovery:** After late night, keep next wake time stable (no extreme sleep-in).
- **Daily check-in:** "Did I stay inside my bedtime guardrail?"
- **7-day progress:** More stable sleep timing and morning energy.

### 10) Mini Body Downshift
- **When to recommend:** Physiological arousal at bedtime.
- **Minimum:** 2 minutes of long-exhale breathing.
- **Standard:** 5 minutes breathing or gentle stretch.
- **Advanced:** 8-minute scripted downshift routine.
- **Failure recovery:** If mind races, restart with one minute only.
- **Daily check-in:** "Did I downshift my body before bed?"
- **7-day progress:** Easier transition from activation to rest.

## Pillar 3: Start With Purpose

### 1) Fixed Wake Anchor
- **When to recommend:** Irregular wake times and social jetlag.
- **Minimum:** Wake within +/- 45 minutes daily.
- **Standard:** Wake within +/- 30 minutes.
- **Advanced:** +/- 20 minutes on weekdays and weekends.
- **Failure recovery:** After poor night, keep anchor and use afternoon light/movement.
- **Daily check-in:** "Did I protect my wake anchor today?"
- **7-day progress:** Stronger circadian stability and earlier daytime alertness.

### 2) Morning Light Dose
- **When to recommend:** Groggy mornings, delayed sleep phase.
- **Minimum:** 5-10 minutes outdoor light within 60 minutes of waking.
- **Standard:** 10-20 minutes most days.
- **Advanced:** 20 minutes + brief walk.
- **Failure recovery:** If morning missed, take midday outdoor light and reset next day.
- **Daily check-in:** "Did I get morning light early enough?"
- **7-day progress:** Better daytime rhythm and evening sleep drive timing.

### 3) Water Before Phone
- **When to recommend:** Reactive phone-first mornings.
- **Minimum:** Drink water before first app.
- **Standard:** Water + 60 seconds of standing stretch.
- **Advanced:** Water + stretch + 2 mindful breaths at window.
- **Failure recovery:** If phone first, do sequence immediately after noticing.
- **Daily check-in:** "Did I start in my body before my feed?"
- **7-day progress:** More intentional morning onset.

### 4) 3-Line Morning Intent
- **When to recommend:** Drifting through day without priorities.
- **Minimum:** Write today's one essential outcome.
- **Standard:** Outcome + why + first action.
- **Advanced:** Add "if-then" for likely obstacle.
- **Failure recovery:** If missed morning, do it at first break.
- **Daily check-in:** "Did I name the day before the day named me?"
- **7-day progress:** Clearer direction and less scatter.

### 5) No-Scroll First 15
- **When to recommend:** Habitual doomscroll on wake.
- **Minimum:** Delay social/news apps for 15 minutes.
- **Standard:** 30 minutes.
- **Advanced:** 60 minutes with app lock.
- **Failure recovery:** If broken, close app and do one grounding action.
- **Daily check-in:** "Did I keep the first minutes for myself?"
- **7-day progress:** Lower morning anxiety and attention fragmentation.

### 6) Micro Movement Start
- **When to recommend:** Morning lethargy and stiffness.
- **Minimum:** 3 minutes of movement after waking.
- **Standard:** 5-7 minutes of brisk movement.
- **Advanced:** 10 minutes including light cardio.
- **Failure recovery:** If skipped, do 3 minutes before lunch.
- **Daily check-in:** "Did I activate my body early today?"
- **7-day progress:** Faster alertness and improved energy slope.

### 7) Protein-First Breakfast Decision
- **When to recommend:** Mid-morning crash and cravings.
- **Minimum:** Choose a protein-containing first meal.
- **Standard:** Protein + fiber combo.
- **Advanced:** Pre-plan two go-to options for workdays.
- **Failure recovery:** If breakfast missed, protein-forward first snack.
- **Daily check-in:** "Did my first meal support stable energy?"
- **7-day progress:** Fewer spikes/crashes, steadier focus.

### 8) First Deep Work Block
- **When to recommend:** Constantly reactive workdays.
- **Minimum:** One 15-minute focused block before meetings/messages.
- **Standard:** 25-minute block.
- **Advanced:** 45-minute block with notifications off.
- **Failure recovery:** If interrupted, restart a 10-minute micro-block.
- **Daily check-in:** "Did I do one meaningful block early?"
- **7-day progress:** More high-value output before noon.

### 9) Morning Environment Reset
- **When to recommend:** Visual clutter creates cognitive clutter.
- **Minimum:** 2-minute desk reset before work start.
- **Standard:** Clear desk + open only needed tabs/apps.
- **Advanced:** Full ritual with water, notebook, and task card.
- **Failure recovery:** If forgotten, reset before next task switch.
- **Daily check-in:** "Did I reset my space for focus?"
- **7-day progress:** Quicker task entry and less friction.

### 10) Commute Purpose Cue
- **When to recommend:** Transition stress between home/work roles.
- **Minimum:** One sentence intention during commute/start walk.
- **Standard:** Intention + one boundary for the day.
- **Advanced:** Add end-of-day closure intention.
- **Failure recovery:** If missed, run cue at first doorway transition.
- **Daily check-in:** "Did I cross into the day with purpose?"
- **7-day progress:** Better role transitions and reduced mental carryover.

## Pillar 4: Train the Future Body

### 1) Daily Movement Floor
- **When to recommend:** Sedentary days, inconsistent exercise.
- **Minimum:** 10 minutes total intentional movement.
- **Standard:** 20 minutes.
- **Advanced:** 30 minutes with moderate intensity segments.
- **Failure recovery:** If evening and low energy, do 5-minute walk now.
- **Daily check-in:** "Did I hit my movement floor?"
- **7-day progress:** Less inactivity and better sleep pressure.

### 2) Step Baseline +10%
- **When to recommend:** Low daily activity baseline.
- **Minimum:** Track current average steps for 2 days.
- **Standard:** Increase by ~10% for remaining days.
- **Advanced:** Maintain +10% and add one brisk interval/day.
- **Failure recovery:** If low-step day, add one 8-minute walk.
- **Daily check-in:** "Did I move a little more than my baseline?"
- **7-day progress:** Practical activity gain without burnout.

### 3) Movement Snacks Each 90 Minutes
- **When to recommend:** Long seated work blocks.
- **Minimum:** 1-minute stand/walk/mobility every 90 minutes.
- **Standard:** 2-minute snack.
- **Advanced:** 3-minute brisk stair/walk snack.
- **Failure recovery:** If several missed, do one 5-minute reset snack.
- **Daily check-in:** "Did I break up long sitting today?"
- **7-day progress:** Reduced stiffness and better daytime alertness.

### 4) Two Strength Micro-Sessions
- **When to recommend:** No resistance training habit.
- **Minimum:** 2 sessions/week, 8 minutes each (bodyweight).
- **Standard:** 2 sessions/week, 15 minutes.
- **Advanced:** 3 sessions/week, 20 minutes progressive.
- **Failure recovery:** Missed session becomes 6-minute version next day.
- **Daily check-in:** "Did I build future-body strength this week?"
- **7-day progress:** Early strength momentum and confidence.

### 5) Post-Meal Walk
- **When to recommend:** Afternoon/evening sluggishness.
- **Minimum:** 5-minute walk after one meal/day.
- **Standard:** 10-minute walk after dinner.
- **Advanced:** 10-minute walk after two meals.
- **Failure recovery:** If not possible outdoors, indoor steps for 4 minutes.
- **Daily check-in:** "Did I walk after at least one meal?"
- **7-day progress:** Better energy regulation and digestion comfort.

### 6) Cardio Pulse Rep
- **When to recommend:** Low cardiovascular challenge.
- **Minimum:** 3 x 1-minute brisk intervals in a walk.
- **Standard:** 5 x 1-minute intervals.
- **Advanced:** 8 x 1-minute intervals with easy recovery.
- **Failure recovery:** If skipped, do 2 intervals during next walk.
- **Daily check-in:** "Did I raise my heart rate briefly today?"
- **7-day progress:** Improved fitness signal with low time cost.

### 7) Mobility Before Bed (Light)
- **When to recommend:** Physical tension at night.
- **Minimum:** 3 minutes gentle mobility/stretching.
- **Standard:** 6 minutes guided routine.
- **Advanced:** 10 minutes with breath-led pacing.
- **Failure recovery:** If too late, do 90 seconds neck/hip release.
- **Daily check-in:** "Did I release body tension tonight?"
- **7-day progress:** Less somatic restlessness near bedtime.

### 8) Weekend Anchor Session
- **When to recommend:** Weekday-only consistency.
- **Minimum:** One 20-minute planned weekend session.
- **Standard:** One 30-minute session.
- **Advanced:** One 40-minute session + recovery walk.
- **Failure recovery:** Split into two 10-minute blocks same day.
- **Daily check-in:** "Did I protect a weekend body investment?"
- **7-day progress:** Better habit continuity across week.

### 9) Equipment-in-View Rule
- **When to recommend:** Forgetting planned workouts.
- **Minimum:** Put shoes/band in visible path each evening.
- **Standard:** Stage full mini-workout setup.
- **Advanced:** Setup + calendar cue + backup location.
- **Failure recovery:** If not staged, stage immediately when noticed.
- **Daily check-in:** "Did my environment remind me to move?"
- **7-day progress:** More sessions started with less negotiation.

### 10) Recovery Day Still Moves
- **When to recommend:** On/off cycles from overtraining mindset.
- **Minimum:** On low-energy days, 10-minute easy walk only.
- **Standard:** Walk + 3-minute mobility.
- **Advanced:** Walk + mobility + hydration target.
- **Failure recovery:** If full rest day happened, next day starts with minimum.
- **Daily check-in:** "Did I keep movement alive on hard days?"
- **7-day progress:** Fewer zero-movement days; better sustainability.

## Pillar 5: Food as Medicine

### 1) Protein + Fiber First Meal
- **When to recommend:** Energy crashes and frequent snacking.
- **Minimum:** Include one protein source at first meal.
- **Standard:** Protein + fiber source.
- **Advanced:** Pre-commit exact weekday first-meal template.
- **Failure recovery:** If meal was low-quality, next intake is protein+fiber.
- **Daily check-in:** "Did my first meal stabilize me?"
- **7-day progress:** More stable energy and fewer urgent cravings.

### 2) Emergency Meal Pair
- **When to recommend:** Busy-day takeout chaos.
- **Minimum:** Pick two 5-minute fallback meals.
- **Standard:** Stock ingredients for both.
- **Advanced:** Add one portable backup snack kit.
- **Failure recovery:** If day derails, use one fallback without guilt.
- **Daily check-in:** "Did I use or maintain my fallback plan?"
- **7-day progress:** Less decision fatigue and late-night overeating.

### 3) Hydration Anchor
- **When to recommend:** Mistaking thirst for hunger/fatigue.
- **Minimum:** One glass at wake + one at lunch.
- **Standard:** Add one in afternoon.
- **Advanced:** Pre-fill bottle with time markers.
- **Failure recovery:** If behind, drink one glass before next snack.
- **Daily check-in:** "Did I hit my hydration anchors?"
- **7-day progress:** Better energy and clearer appetite signals.

### 4) Late-Night Eating Guardrail
- **When to recommend:** Regular post-dinner grazing.
- **Minimum:** Kitchen closes 90 minutes before bed.
- **Standard:** 120-minute close.
- **Advanced:** 150-minute close with planned light option if needed.
- **Failure recovery:** If eating late, keep portion small and intentional.
- **Daily check-in:** "Did I keep my evening eating guardrail?"
- **7-day progress:** Reduced late-night intake and sleep disruption.

### 5) Plate Upgrade Rule
- **When to recommend:** Processed default meals.
- **Minimum:** Add one whole-food element to one meal.
- **Standard:** Upgrade two meals/day.
- **Advanced:** Build one full "anchor plate" daily.
- **Failure recovery:** If meal was low quality, upgrade next plate only.
- **Daily check-in:** "Did I improve at least one plate today?"
- **7-day progress:** Better nutrient density without rigid dieting.

### 6) Smart Caffeine Use
- **When to recommend:** Afternoon slump and poor sleep.
- **Minimum:** Keep caffeine to planned window only.
- **Standard:** Max 1-2 planned doses before cutoff.
- **Advanced:** Lower dose on stress days to reduce jitter loop.
- **Failure recovery:** If extra dose taken, protect wind-down and light hygiene.
- **Daily check-in:** "Did I use caffeine as a tool, not a reflex?"
- **7-day progress:** More predictable energy and sleep readiness.

### 7) Snack Decision Script
- **When to recommend:** Mindless snacking.
- **Minimum:** Before snack ask: "Hungry, tired, stressed, or bored?"
- **Standard:** If not hungry, do 3-minute alternative first.
- **Advanced:** Log trigger pattern once/day.
- **Failure recovery:** If mindless snack happened, pause and reset next choice.
- **Daily check-in:** "Did I make one snack decision consciously?"
- **7-day progress:** More intentional eating and fewer autopilot snacks.

### 8) Dinner Simplicity Protocol
- **When to recommend:** Evening food overwhelm.
- **Minimum:** Rotate 2 simple dinners for 7 days.
- **Standard:** 3 rotating dinners.
- **Advanced:** Include one prep-ahead component.
- **Failure recovery:** If plan fails, use emergency meal pair.
- **Daily check-in:** "Did I keep dinner simple enough to execute?"
- **7-day progress:** Reduced evening stress and better consistency.

### 9) Alcohol Boundary Micro-Plan
- **When to recommend:** Sleep affected by late alcohol use.
- **Minimum:** Define alcohol-free nights for the week.
- **Standard:** Stop alcohol at least 3 hours pre-bed.
- **Advanced:** Alternate each drink with water + stop-rule cue.
- **Failure recovery:** If boundary broken, hydrate and resume plan next day.
- **Daily check-in:** "Did my alcohol choices support tonight's sleep?"
- **7-day progress:** Better second-half sleep quality and morning recovery.

### 10) 60-Second Food Log
- **When to recommend:** Low awareness of patterns.
- **Minimum:** Log first meal and last meal times.
- **Standard:** Add one note on energy after lunch.
- **Advanced:** Add one trigger tag for unplanned eating.
- **Failure recovery:** If missed logging, do one end-of-day reconstruction.
- **Daily check-in:** "Did I track enough to learn one pattern?"
- **7-day progress:** Better self-awareness and easier targeted adjustments.

## Pillar 6: Kill Distractions

- **Standard:** Two 25-minute blocks.
- **Advanced:** Two 45-minute blocks with full focus mode.
- **Failure recovery:** If interrupted, restart a 10-minute micro-block.

### 2) Single-Tab Rule
- **When to recommend:** Browser-tab sprawl.
- **Minimum:** Keep only one active task tab set visible.
- **Standard:** Close unrelated tabs before each block.
- **Advanced:** Use separate profiles/workspaces for roles.
- **Failure recovery:** Tab overload -> 60-second tab reset.
- **Daily check-in:** "Did my screen match one priority at a time?"

### 3) Message Windowing

- **Advanced:** 2 windows + urgent filter rule.
- **Failure recovery:** If reactive checking happens, return to next planned window.

### 4) Phone Out-of-Reach Sprint

- **Minimum:** Phone out of reach for one 20-minute sprint.

- **Advanced:** Use another room for first morning sprint.

### 5) Start Cue Script

- **Standard:** Phrase + open task doc + timer.
- **Advanced:** Add pre-block intention and end note.

### 6) Distraction Capture Pad

### 7) Social App Friction

### 9) Evening Info Cap

- **Failure recovery:** If cap exceeded, close apps and start mini wind-down.

- **7-day progress:** Less nighttime cognitive noise.

### 10) Weekly Digital Reset (within 7-day cycle)
- **When to recommend:** Chronic digital clutter.
- **Minimum:** One 10-minute cleanup session.
- **Standard:** Clean notifications, home screen, and top feeds.
- **Advanced:** Rebuild digital defaults around priorities.
- **Failure recovery:** If missed, do 5-minute version next morning.
- **Daily check-in:** "Did I maintain a cleaner digital environment?"
- **7-day progress:** Easier focus due to better default architecture.

## Pillar 7: End Isolation

### 1) One Intentional Reach-Out
- **When to recommend:** Emotional withdrawal and stress isolation.
- **Minimum:** Send one genuine check-in message daily.
- **Standard:** Message + one follow-up question.
- **Advanced:** Include short voice note or call invite.
- **Failure recovery:** If missed, send a 2-line message now.
- **Daily check-in:** "Did I reach for a person, not just a screen?"
- **7-day progress:** Increased felt connection and support access.

### 2) 10-Minute Human Contact
- **When to recommend:** Remote work loneliness.
- **Minimum:** 10 minutes of real conversation/day.
- **Standard:** 15 minutes with active listening.
- **Advanced:** 20 minutes with shared activity (walk/coffee).
- **Failure recovery:** If no live contact, schedule one before bedtime.
- **Daily check-in:** "Did I have real human contact today?"
- **7-day progress:** Lower isolation drift and better mood regulation.

### 3) Ask Before Crisis Protocol
- **When to recommend:** Only asking for help when overwhelmed.
- **Minimum:** Make one small ask this week.
- **Standard:** Two small asks across 7 days.
- **Advanced:** Ask + share context and preferred help format.
- **Failure recovery:** If avoidant, send prewritten one-sentence ask.
- **Daily check-in:** "Did I practice asking early, not late?"
- **7-day progress:** Stronger help-seeking confidence.

### 4) Shared Meal/Break Rep
- **When to recommend:** Eating/working alone every day.
- **Minimum:** One shared meal or break this week.
- **Standard:** Two shared breaks.
- **Advanced:** Three brief shared moments.
- **Failure recovery:** If canceled, replace with 10-minute call.
- **Daily check-in:** "Did I share at least one routine moment?"
- **7-day progress:** More social rhythm in weekly structure.

### 5) Appreciation Message Habit
- **When to recommend:** Relationship maintenance neglect.
- **Minimum:** One appreciation text every other day.
- **Standard:** One daily short appreciation.
- **Advanced:** Appreciation + specific example of impact.
- **Failure recovery:** If missed, send one before sleep.
- **Daily check-in:** "Did I express appreciation today?"
- **7-day progress:** Warmer relational tone and reciprocity.

### 6) Low-Energy Social Option List
- **When to recommend:** Socializing feels too effortful when tired.
- **Minimum:** Create 3 low-energy connection options.
- **Standard:** Use one option on a hard day.
- **Advanced:** Share list with friend/partner for easier coordination.
- **Failure recovery:** If isolating, pick the easiest option immediately.
- **Daily check-in:** "Did I use a low-energy connection option when needed?"
- **7-day progress:** Less all-or-nothing social behavior.

### 7) Community Touchpoint
- **When to recommend:** Lack of belonging beyond close circle.
- **Minimum:** One weekly community interaction (group/forum/class).
- **Standard:** Two touchpoints.
- **Advanced:** One contribution (comment/help/share resource).
- **Failure recovery:** If missed, do 5-minute participation action.
- **Daily check-in:** "Did I touch my wider community this week?"
- **7-day progress:** Early increase in belonging cues.

### 8) Co-Work or Co-Walk Slot
- **When to recommend:** Motivation dips when alone.
- **Minimum:** One 20-minute co-work/co-walk session.
- **Standard:** Two sessions.
- **Advanced:** Three sessions with pre-set objective.
- **Failure recovery:** If partner unavailable, use body-double virtual option.
- **Daily check-in:** "Did I use social structure for momentum?"
- **7-day progress:** Better consistency through co-regulation.

### 9) Stress Signal Share
- **When to recommend:** Masking stress until shutdown.
- **Minimum:** Share one honest stress signal with trusted person.
- **Standard:** Share + name one needed support.
- **Advanced:** Set a brief follow-up check-in.
- **Failure recovery:** If withheld, send a short update message tonight.
- **Daily check-in:** "Did I let someone know where I really am?"
- **7-day progress:** Reduced internal load and better emotional regulation.

### 10) Isolation Interrupt Rule
- **When to recommend:** Repeated "disappear" pattern after hard days.
- **Minimum:** If urge to isolate > 1 hour, send one connection ping.
- **Standard:** Ping + 5-minute call invite.
- **Advanced:** Ping + schedule tomorrow micro-connection.
- **Failure recovery:** If isolation already happened, reconnect next morning first hour.
- **Daily check-in:** "Did I interrupt isolation quickly?"
- **7-day progress:** Faster recovery from withdrawal episodes.

## Pillar 8: Build Self-Trust

### 1) One Promise Rule
- **When to recommend:** Chronic self-betrayal from overcommitment.
- **Minimum:** Set one promise under 10 minutes/day.
- **Standard:** Keep one promise and log completion.
- **Advanced:** Keep one promise + one optional stretch.
- **Failure recovery:** If missed, shrink tomorrow's promise further.
- **Daily check-in:** "Did I keep today's core promise to myself?"
- **7-day progress:** Stronger identity evidence: "I do what I say."

### 2) Honest Scorecard
- **When to recommend:** Perfectionism or vague self-evaluation.
- **Minimum:** Mark day as kept/partly/missed.
- **Standard:** Add one sentence why.
- **Advanced:** Add one adjustment for tomorrow.
- **Failure recovery:** Missed day still gets logged (no blank days).
- **Daily check-in:** "Did I score myself honestly, not harshly?"
- **7-day progress:** Clearer pattern awareness and less self-deception.

### 3) If-Then Obstacle Plan
- **When to recommend:** Repeated predictable derailers.
- **Minimum:** Write one `if obstacle, then tiny action` plan.
- **Standard:** Two plans for top derailers.
- **Advanced:** Link plans to time/place cues.
- **Failure recovery:** If plan fails, rewrite with smaller action.
- **Daily check-in:** "Did I use my if-then plan when friction hit?"
- **7-day progress:** Better follow-through under stress.

### 4) Non-Zero Day Standard
- **When to recommend:** All-or-nothing collapse.
- **Minimum:** Do at least 1 minute of protocol action daily.
- **Standard:** 3-minute non-zero action.
- **Advanced:** 5-minute action plus brief log.
- **Failure recovery:** If near bedtime, complete a 60-second version.
- **Daily check-in:** "Did I keep the day non-zero?"
- **7-day progress:** Streak resilience and reduced quitting behavior.

### 5) Fast Repair Script
- **When to recommend:** One lapse becomes multi-day lapse.
- **Minimum:** Say/write: "Lapse is data. Next rep now."
- **Standard:** Script + immediate 2-minute repair action.
- **Advanced:** Script + repair + trigger analysis.
- **Failure recovery:** If spiraling, contact accountability partner and do minimum action.
- **Daily check-in:** "Did I repair fast instead of judging long?"
- **7-day progress:** Shorter downtime after misses.

### 6) Win Evidence Log
- **When to recommend:** Negative bias, undercounting progress.
- **Minimum:** Log one kept commitment daily.
- **Standard:** Log two pieces of evidence.
- **Advanced:** Evidence + what enabled it.
- **Failure recovery:** If no obvious win, log one micro-win.
- **Daily check-in:** "What evidence did I earn today?"
- **7-day progress:** Increased confidence grounded in behavior.

### 7) Boundary Kept Rep
- **When to recommend:** Saying yes to everything.
- **Minimum:** Keep one small boundary/day.
- **Standard:** Keep one boundary + communicate it clearly.
- **Advanced:** Keep two boundaries in high-friction contexts.
- **Failure recovery:** If boundary broke, restate it at next opportunity.
- **Daily check-in:** "Did I protect one boundary today?"
- **7-day progress:** Greater agency and reduced resentment.

### 8) Identity Statement Pairing
- **When to recommend:** Motivation tied only to mood.
- **Minimum:** Read one identity line before action ("I am someone who repairs quickly").
- **Standard:** Read + do immediate tiny action.
- **Advanced:** Update identity line from real evidence.
- **Failure recovery:** If action skipped, repeat line and do 60 seconds.
- **Daily check-in:** "Did I act in line with who I'm becoming?"
- **7-day progress:** Stronger identity-action alignment.

### 9) Weekly Micro-Review (Day 7)
- **When to recommend:** Repeating same mistakes without learning.
- **Minimum:** Review: worked / hard / next tiny tweak.
- **Standard:** Add one stop-doing decision.
- **Advanced:** Add next week's protocol selection rule.
- **Failure recovery:** If postponed, do 5-minute voice note review.
- **Daily check-in:** "Did I learn from this week, not just judge it?"
- **7-day progress:** Better adaptation and smarter next cycle.

### 10) Accountability Ping
- **When to recommend:** Low consistency when solo.
- **Minimum:** Send one daily check mark to partner/group.
- **Standard:** Check mark + 1 line reflection.
- **Advanced:** Reflection + tomorrow commitment.
- **Failure recovery:** If missed ping, send late ping with restart note.
- **Daily check-in:** "Did I report in, even on imperfect days?"
- **7-day progress:** Higher consistency through gentle social accountability.

## Simple Recommendation Logic (Optional for Coaching/Automation)
- Recommend **Minimum** when motivation is low, schedule is unstable, or confidence < 6/10.
- Recommend **Standard** when minimum is completed on 4+ of the last 7 days.
- Recommend **Advanced** when standard is completed on 5+ of the last 7 days with low strain.
- After any lapse cluster (2+ missed days), auto-switch to **Minimum + Fast Repair Script** for 48 hours.
